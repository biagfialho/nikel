✨ Sobre o Projeto
Nikel é uma página web desenvolvida com foco nos fundamentos da criação de interfaces modernas. Utilizando HTML, CSS e JavaScript, o projeto explora conceitos essenciais de estruturação, estilização e interatividade, com o objetivo de oferecer uma base sólida para quem está começando no desenvolvimento front-end.

A proposta é simples e direta: construir uma interface limpa, responsiva e funcional, aplicando boas práticas de codificação e organização de arquivos. O projeto serve tanto como exercício prático quanto como referência para estudos iniciais em desenvolvimento web.
